## 一、冲击类故障

> **能量表现**：瞬时宽频激励，能量集中于壳体/缸体的结构固有频率（高频共振区）。特征频率只作为包络谱上的调制频率存在。
>
> **算法搜寻区域**：共振频带（非低频特征频率）。
>
> **检测路径**：带通滤波（共振频带）→ Hilbert包络 → 包络谱 → 在包络谱上找特征频率。

### 1.1 松靴/滑靴脱落

> **文献支撑**：[R1] 分析了松靴故障机理，建立了松靴与壳体轴向振动信号间的映射关系，采用倒频谱提取了 f_p 谐波特征；[R10] 针对松靴微弱特征提出了最小熵反褶积方法。

| 维度 | 内容 |
|------|------|
| **故障部位** | 柱塞球头与滑靴连接松动 |
| **物理机理** | 柱塞球头松动 → 每转一圈滑靴撞击斜盘一次 → 冲击力激发壳体/缸体结构固有频率共振 [R1] |
| **冲击重复频率** | f_p = 172.67 Hz（每转每个柱塞撞击一次，7个柱塞） |
| **高频共振带** | 3~8 kHz（壳体固有频率，取决于结构；20kHz采样率下可用6~7.5kHz） |
| **包络谱特征** | 在包络谱上出现 f_p（172.67 Hz）及其谐波 2f_p、3f_p... [R1] |
| **时域特征** | 峭度显著增大（冲击脉冲）、波峰因子增大 [R1][R10] |
| **角域特征** | TOT路径角域峭度增大（冲击在等角度采样后更明显） |
| **轴间特征** | Z轴峭度远高于X轴（轴向冲击为主） [R1] |
| **当前检测方法** | 高频带能量比 + 包络谱 f_p 幅值比 + 角域峭度 |

### 1.2 轴承外圈故障

> **文献支撑**：[R3] 完成了轴承外圈断裂故障试验，验证了振动信号时频域分析方法的有效性；[R7][R8] 证明 Kurtogram+包络谱是轴承故障诊断的标准方法；[R12] 验证了 Kurtogram 包络解调在滚动轴承中的应用。

| 维度 | 内容 |
|------|------|
| **故障部位** | 滚动轴承外圈滚道点蚀/剥落 [R3] |
| **物理机理** | 外圈固定不动，滚动体经过剥落坑时产生冲击 → 激发轴承外圈固有频率（通常3~8 kHz） [R7][R12] |
| **特征频率** | BPFO = (N_b/2) × f_r × (1 - d/D×cosα)，其中 N_b=滚动体数, d=滚动体直径, D=节圆直径, α=接触角。典型柱塞泵轴承 BPFO ≈ 4~8×f_r |
| **高频共振带** | 3~8 kHz（轴承外圈固有频率） [R7] |
| **包络谱特征** | BPFO 及其谐波，可能伴有转频 f_r 边频 [R8] |
| **时域特征** | 峭度增大、波峰因子增大（但弱于松靴） |
| **注意** | 需要轴承几何参数才能精确计算 BPFO；无参数时可搜索 4~8×f_r 范围内的谐波族 |

### 1.3 轴承内圈故障

> **文献支撑**：[R4] 研究了轴承内圈故障的振动特性；[R8] 证明内圈故障包络谱中特征频率两侧有转频边频是区分内外圈故障的关键。

| 维度 | 内容 |
|------|------|
| **故障部位** | 滚动轴承内圈滚道点蚀/剥落 [R4] |
| **物理机理** | 内圈随轴旋转，剥落坑周期性经过滚动体 → 冲击频率 = BPFI，且受转频调制（内圈旋转） [R8] |
| **特征频率** | BPFI = (N_b/2) × f_r × (1 + d/D×cosα)，典型值 ≈ 5~9×f_r |
| **包络谱特征** | BPFI 及其谐波，且每个谐波两侧有 ±f_r 边频（内圈旋转调制） [R8] |
| **与外圈的区别** | 内圈故障有转频边频，外圈没有 |
| **注意** | 同样需要轴承几何参数 |

### 1.4 气穴溃灭

> **文献支撑**：[R5] 系统研究了轴向柱塞泵空化检测与故障诊断方法；[R6] 综述了空化对振动和噪声的影响机理。柱塞腔空化机理见《高速轴向柱塞泵柱塞腔空化机理研究》(液压与气动, 2022)。

| 维度 | 内容 |
|------|------|
| **故障部位** | 泵吸油腔局部压力低于空气分离压，产生气泡并在高压区溃灭 [R5] |
| **物理机理** | 气泡溃灭产生瞬态宽频冲击 → 能量分布随机，无明确特征频率 [R5][R6] |
| **时域特征** | 峭度增大但无周期性（随机冲击）；与松靴的区别：松靴有 f_p 周期，气穴没有 [R5] |
| **频域特征** | 高频底噪整体抬升（宽频激励） [R5] |
| **包络谱特征** | 无明显的离散频率峰值（区别于松靴的 f_p 谱线） |
| **伴随现象** | 出口压力脉动增大、噪声增大 [R6] |
| **检测难点** | 需与松靴区分——气穴无周期性，松靴有 f_p 周期 |

## 二、低频周期类故障

> **能量表现**：能量直接体现在转频（1×, 2×）或柱塞通过频率（f_r×柱塞数）及其谐波上。
>
> **算法搜寻区域**：低频 FFT 谱。
>
> **检测路径**：直接 FFT → 在频谱上找特征频率及其谐波。

### 2.1 配流盘磨损

> **文献支撑**：[R2] 提出了基于CEEMDAN与t-SNE的配流盘磨损诊断方法；[R9] 建立了配流副动力学模型，分析了端面磨损对振动响应的影响。

| 维度 | 内容 |
|------|------|
| **故障部位** | 配流盘端面（与缸体接触面）磨损 [R2] |
| **物理机理** | 端面磨损 → 高低压腔串通 → 内泄漏增大 → 流量/压力脉动增大 [R9] |
| **特征频率** | f_p（172.67 Hz）及其谐波——配流盘每转完成7次高低压切换 [R2] |
| **频域特征** | f_p 及谐波幅值均匀增大（区别于柱塞磨损的特定阶次增大） [R2] |
| **低频带** | 10~250 Hz（转频谐波区）+ 150~700 Hz（柱塞频率区） |
| **重点关注阶次** | 1~7×f_p（配流盘磨损影响所有柱塞，所以谐波全面抬升） |
| **与柱塞磨损的区别** | 配流盘磨损 → 所有柱塞受影响 → 谐波全面抬升；柱塞磨损 → 单个柱塞 → 特定阶次突出 |

### 2.2 柱塞-缸孔磨损

> **文献支撑**：[R2] 对柱塞磨损的振动特征进行了时频域分析；[R3] 完成了单柱塞磨损故障试验。

| 维度 | 内容 |
|------|------|
| **故障部位** | 柱塞外圆与缸孔内壁配合面磨损 [R2][R3] |
| **物理机理** | 间隙增大 → 内泄漏增大 → 泄漏流量脉动 → 柱塞频率谐波增大 [R2] |
| **特征频率** | f_p（172.67 Hz）及其谐波，以 1f_p、2f_p、3f_p 为主 [R2] |
| **频域特征** | f_p 谐波增大，但以低阶谐波为主（1f_p~3f_p），高阶谐波增幅递减 |
| **低频带** | 50~500 Hz（泄漏脉动区）+ 150~900 Hz（柱塞频率区） |
| **重点关注阶次** | 7、14、21（即 1f_p、2f_p、3f_p） |

### 2.3 斜盘磨损/异常

> **文献支撑**：[R4] 研究了斜盘磨损的振动特性；[R11] 建立了斜盘局部缺陷的动力学模型，分析了耦合振动特性。

| 维度 | 内容 |
|------|------|
| **故障部位** | 斜盘摩擦面（与滑靴接触面）磨损，或斜盘倾角控制异常 [R4][R11] |
| **物理机理** | 斜盘面不平 → 柱塞运动不平稳 → 轴向力周期性波动 → 转频及其谐波增大 [R11] |
| **特征频率** | f_r（24.67 Hz）及其谐波为主，f_p 也会受影响 [R4] |
| **频域特征** | 低频转频谐波（1×、2×、3×f_r）增大 |
| **与配流盘的区别** | 斜盘磨损主要影响转频谐波（f_r），配流盘磨损主要影响柱塞频率谐波（f_p） |

### 2.4 缸体/主轴不平衡

| 维度 | 内容 |
|------|------|
| **故障部位** | 缸体或主轴的质量不平衡 |
| **物理机理** | 质量偏心 → 离心力 → 转频振动增大 |
| **特征频率** | 1×f_r（24.67 Hz），径向振动为主 |
| **频域特征** | 1×f_r 幅值显著增大，2×f_r 可能有小幅度增大 |
| **轴间特征** | X/Y轴（径向）振动远大于Z轴（轴向） |
| **与其他故障的区别** | 纯转频增大，无 f_p 谐波 |

### 2.5 联轴器不对中

| 维度 | 内容 |
|------|------|
| **故障部位** | 电机与泵之间的联轴器安装不对中 |
| **物理机理** | 角度不对中 → 转频的2倍频激励；平行不对中 → 径向力 |
| **特征频率** | 2×f_r（49.34 Hz）为主，1×f_r 也有 |
| **频域特征** | 2×f_r > 1×f_r（角度不对中的典型模式） |
| **轴间特征** | 径向（X/Y）180°相位差（角度不对中） |

### 2.6 柱塞卡滞

| 维度 | 内容 |
|------|------|
| **故障部位** | 柱塞在缸孔内卡住，不能自由往复运动 |
| **物理机理** | 柱塞卡死 → 该柱塞不供油 → 流量脉动周期性缺失 → f_p 谐波变化 |
| **特征频率** | f_p 谐波结构变化（某些阶次缺失或异常） |
| **与磨损的区别** | 磨损是谐波全面增大；卡滞是谐波结构异常（缺失或不规则） |
| **检测难点** | 需要对比正常信号的谐波结构，不能只看幅值 |

## 三、均匀磨损类故障

> **能量表现**：无突出冲击尖峰，出现密集边频带、底噪整体抬升，或出现分数谐波。
>
> **算法搜寻区域**：边频带密度、功率谱整体统计特性。
>
> **检测路径**：频谱统计分析（谱熵、谱质心）+ 边频带密度 + RMS趋势。

### 3.1 配流盘均匀磨损

| 维度 | 内容 |
|------|------|
| **故障部位** | 配流盘端面均匀磨损（区别于局部磨损的2.1） |
| **物理机理** | 端面均匀变薄 → 高低压腔密封带变窄 → 整体泄漏增大 → 底噪抬升 |
| **频域特征** | f_p 谐波不一定突出，但全频带底噪抬升、谱熵增大 |
| **与局部磨损的区别** | 局部磨损 → f_p 谐波突出；均匀磨损 → 底噪全面抬升，无突出频率 |

### 3.2 柱塞-缸孔均匀磨损

| 维度 | 内容 |
|------|------|
| **故障部位** | 所有柱塞与缸孔的配合间隙均匀增大 |
| **物理机理** | 间隙全面增大 → 泄漏全面增大 → 密集边频带、分数谐波出现 |
| **频域特征** | f_p 周围出现密集边频带（间距 = f_r），分数谐波（如 0.5×f_p） |
| **统计特征** | 谱熵增大（能量更分散）、谱质心偏移 |

## 五、算法自检清单

每引入一个处理步骤，必须能回答以下问题：

| 步骤 | 必须回答的"为什么" |
|------|-------------------|
| 去均值/去趋势 | 消除直流分量对谱分析的影响，防止低频趋势项淹没故障特征 |
| Kurtogram 选频带 | 峭度对冲击脉冲最敏感，自适应找到信噪比最高的共振频带 |
| 带通滤波 | 滤除无关激励，只保留被故障冲击激起的调制信号 |
| Hilbert 包络 | 将高频共振带中的时域波形起伏提取出来，包络的周期性 = 故障特征频率 |
| 包络谱 FFT | 在包络谱上找特征频率——此刻才能做"去正确的地方做对比" |
| 信噪比计算 | 特征频率幅值/背景噪声，量化冲击周期能量相对于随机噪声的强度 |
| 谐波+边频验证 | 反映故障周期性结构的完整性，排除随机冲击假象 |

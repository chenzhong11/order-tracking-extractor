## 1. 仿真信号有效性验证

> **目的**: 用已知参数的合成信号，验证算法能否精确恢复特征频率。
>
> **对应图**: `fig1_signal_validation.png`

### 1.1 测试方法

合成信号构成：
- **正常信号** = 转频脉动 ($f_r$=24.67Hz) + 柱塞频率脉动 ($f_p$=172.67Hz) + 高斯白噪声
- **故障信号** = 正常信号 + 指数衰减冲击脉冲串（周期 = $1/f_p$，模拟松靴撞击）

### 1.2 验证内容

| 子图 | 验证内容 | 判定标准 |
|------|----------|----------|
| (a) 时域波形 | 冲击脉冲是否可见 | 故障信号在 $f_p$ 周期位置出现明显尖峰 |
| (b) FFT 频谱 | $f_r$, $f_p$, $2f_p$ 谱线是否准确 | 频率误差 < 1% |
| (c) 包络谱 | 共振解调能否恢复 $f_p$ 冲击重复频率 | 包络谱中 $f_p$ 处幅值显著高于噪声底 |
| (d) TOT 阶次谱 | 阶次分析能否定位到 7 阶和 14 阶 | 7 阶和 14 阶幅值在故障时显著增大 |

### 1.3 结论

✅ 合成信号白盒测试通过：算法能精确恢复 $f_r$, $f_p$, $2f_p$，包络谱能检出 $f_p$ 冲击重复频率，TOT 阶次谱能定位 7 阶和 14 阶。

## 2. 阶次分析算法有效性

> **目的**: 对比 COT（无转速计）和 TOT（有转速计）两种阶次分析方法的精度和故障检出能力。
>
> **对应图**: `fig2_order_tracking_validation.png`

### 2.1 测试方法

- **数据**: 真实柱塞泵正常/故障振动信号（X轴，20kHz采样，7柱塞，1480RPM）
- **COT**: 通过 STFT 脊线估计瞬时频率
- **TOT**: 用恒定转频（24.67Hz）作为转速参考

### 2.2 验证内容

| 子图 | 验证内容 | 判定标准 |
|------|----------|----------|
| (a) 转频估计 | COT/TOT 估计转频与真实值对比 | TOT 误差=0%，COT 误差<2% |
| (b) 阶次幅值 | COT/TOT 阶次幅值一致性 | 两者趋势一致，TOT 更稳定 |
| (c) COT 检出比值 | 故障/正常的阶次幅值比 | 7 阶和 14 阶比值 > 2.0x |
| (d) TOT 检出比值 | 同上 | 比值更稳定，受 STFT 参数影响小 |

### 2.3 结论

| 指标 | COT | TOT |
|------|-----|-----|
| 转频估计误差 | 1.46% | 0.00% |
| 7 阶检出比值 | 3.11x | 3.00x |
| 14 阶检出比值 | 6.12x | 3.00x |
| 稳定性 | 受 STFT 窗长影响 | 稳定 |

✅ **TOT 优于 COT**：转频精度更高、阶次比值更稳定。COT 在无转速信号时仍可用，但建议窗长 ≥ 2048。

## 3. 统计阈值对比验证

> **目的**: 对比固定比值阈值、3σ 统计阈值、P95 统计阈值三种判定方式的检出效果。
>
> **对应图**: `fig3_threshold_comparison.png`

### 3.1 测试方法

- **校准**: 从正常信号中用滑动窗口切分 194 段，每段提取 106 个特征，计算统计阈值
- **测试**: 同一组故障数据分别用三种模式诊断

### 3.2 三种模式

| 模式 | 阈值来源 | 特点 |
|------|----------|------|
| 固定比值 | 基线值 × 倍数（2.0x/1.5x） | 简单直接，但缺乏统计依据 |
| 3σ | μ + 3σ（基于正常样本分布） | 99.7% 置信度，对偏态分布可能过严 |
| P95 | 95% 分位数 | 对偏态分布更稳健 |

### 3.3 结果对比

| 故障类型 | 固定比值 | 3σ | P95 |
|----------|----------|-----|-----|
| 松靴/滑靴脱落 | ✅ 2.0 | ✅ 0.5 | ✅ 0.5 |
| 配流盘磨损 | ✅ 2.0 | ✗ 0.5 | ✗ 0.5 |
| 柱塞-缸孔磨损 | ✅ 2.0 | ✗ 0.0 | ✗ 0.0 |
| 斜盘磨损 | ✗ 0.0 | ✗ 0.0 | ✗ 0.0 |
| 轴承外圈故障 | ✅ 1.0 | ✗ 0.0 | ✗ 0.0 |
| 轴承内圈故障 | ✅ 1.0 | ✗ 0.0 | ✗ 0.0 |
| 气穴溃灭 | ✗ 0.0 | ✗ 0.0 | ✗ 0.0 |

### 3.4 结论

- **松靴**（峭度特征突出）三种模式都能检出，说明该特征确实显著
- **统计阈值比固定比值更保守**：3σ/P95 仅检出松靴，固定比值检出 5 种
- **需要更多正常样本**来稳定统计阈值（当前只有 1 组正常数据，切分为 194 段伪样本）
- 固定比值阈值（2.0x/1.5x）在当前数据上表现良好，但缺乏统计泛化能力

## 4. 故障诊断综合报告

> **目的**: 用真实数据展示完整的诊断流程和结果。
>
> **对应图**: `fig4_diagnostic_report.png`

### 4.1 报告内容

| 区域 | 内容 |
|------|------|
| 上排左 | 时域波形对比（正常 vs 故障），标注峭度和 RMS |
| 上排右 | 故障概率排序（置信度条形图） |
| 中排 | FFT 频谱（非均匀横轴：0~600Hz 精细 + 600~8kHz 压缩），峰值数值标注 |
| 下排左 | 包络谱（6~7.5kHz 共振解调），故障填充 + 峰值标注 |
| 下排右 | 诊断依据（触发故障、关键频率、触发特征） |
| 底排 | 设备参数 + 诊断结论 |

### 4.2 诊断结果

```
设备: 7柱塞, 1480 RPM, 采样率 20kHz
转频 f_r=24.67Hz, 柱塞通过频率 f_p=7×f_r=172.67Hz

诊断结论: 检测到多种可能故障，最可能是: 松靴/滑靴脱落
检测到: 松靴/滑靴脱落, 配流盘磨损, 柱塞-缸孔磨损, 轴承外圈故障, 轴承内圈故障
```

### 4.3 判断依据

| 故障 | 主触发特征 | 比值 | 阈值 | 机理 |
|------|-----------|------|------|------|
| 松靴/滑靴脱落 | Z轴峭度 | 3.45x | 2.0x | 柱塞球头松动，周期性撞击斜盘，冲击力激发结构共振 |
| | X轴角域峭度 | 9.86x | 1.5x | |
| 配流盘磨损 | 0~100Hz频带能量 | 156x | 2.0x | 配流盘端面磨损，内泄漏增大，流量/压力脉动增大 |
| | 7阶次幅值 | 14x | 1.5x | |
| 柱塞-缸孔磨损 | 7阶次幅值 | 14x | 1.5x | 柱塞与缸孔间隙增大，内泄漏增大，流量脉动加剧 |
| | 14阶次幅值 | 6.4x | 1.5x | |
| 轴承外圈故障 | Z轴峭度 | 3.45x | 2.0x | 外圈滚道点蚀/剥落，滚动体经过剥落坑产生冲击 |
| 轴承内圈故障 | Z轴峭度 | 3.45x | 2.0x | 内圈滚道点蚀/剥落，冲击频率受转频调制 |

## 5. 故障-信号映射表

> 详见 `docs/故障-信号映射表.md`

| 故障类型 | 类别 | 核心特征频率 | 搜寻区域 | 关键区分点 |
|----------|------|-------------|----------|-----------|
| 松靴/滑靴 | 冲击 | $f_p$=172.67Hz | 包络谱（高频共振带） | 包络谱 $f_p$ 谐波，有周期性 |
| 轴承外圈 | 冲击 | BPFO≈4~8×$f_r$ | 包络谱（高频共振带） | 无转频边频 |
| 轴承内圈 | 冲击 | BPFI≈5~9×$f_r$ | 包络谱（高频共振带） | 有转频边频 |
| 气穴溃灭 | 冲击 | 无离散频率 | 高频底噪 | 随机冲击，无周期性 |
| 配流盘磨损 | 低频周期 | $f_p$ 谐波 | 低频 FFT | 谐波全面抬升 |
| 柱塞磨损 | 低频周期 | $f_p$ 谐波 | 低频 FFT | 低阶谐波为主 |
| 斜盘磨损 | 低频周期 | $f_r$ 谐波 | 低频 FFT | 转频谐波为主 |
